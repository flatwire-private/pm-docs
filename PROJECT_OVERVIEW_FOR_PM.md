# MyISHRM Project Overview

**For:** Project Manager / Client Presentation
**Date:** January 9, 2026

---

## What is MyISHRM?

A modern Student Information System (SIS) for ISHRM that handles the complete student lifecycle - from application to graduation.

**Replaces:** The old portal at coursesishrm.edu.ph

---

## What It Does (Key Features)

- **Enrollment** - Online application with payment verification
- **Grades** - Faculty submits, students view
- **Payments** - Track tuition, verify receipts
- **Documents** - Request TOR, certificates online
- **Messaging** - Internal communication between users

---

## The Enrollment Flow (Simple Version)

```
Applicant Applies → Admission Approves → Applicant Pays → Accounting Verifies → Registrar Enrolls
```

Each step has proper verification before moving to the next.
